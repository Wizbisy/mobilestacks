## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Install dependencies** with `npm install`.
3. **Add or update tasks** in the `src/tasks/` directory.
4. **Write tests** for new features or bug fixes.
5. **Run lint and tests** before submitting:
   - `npm run lint`
   - `npm test`
6. **Open a Pull Request** with a clear description.

## Code Style
- Use TypeScript and follow the existing code patterns.
- Keep CLI output user-friendly and secure (no secrets in logs).
- Document new tasks and config options in the README.

## Reporting Issues
- Use [GitHub Issues](https://github.com/Wizbisy/mobilestacks/issues) for bugs, feature requests, or questions.

## Community
- Be respectful and constructive.
- Help others and review PRs if you can.
